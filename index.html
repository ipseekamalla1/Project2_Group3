Bar Chart Visualization

Top 10 Countries by COVID-19 Cases

This bar chart shows the top 10 countries with the highest number of COVID-19 cases. The height of each bar represents the total number of cases in that country.

Hover over a bar to see the exact number of cases. The bar will change color, and a tooltip will display detailed information.

Pie Chart Visualization

Global COVID-19 Distribution

This pie chart illustrates the global distribution of COVID-19 cases, showing the proportion of active cases, recoveries, and deaths worldwide of 8 countries

Click a slice to expand it and highlight the selected country.

Scatter Plot Visualization

Cases vs Deaths Correlation

This scatter plot examines the relationship between the number of COVID-19 cases and deaths across different countries. Each point represents a country.

Zoom over a data point to view detailed information about a country's cases and deaths. Click on a point to highlight it and see a visual emphasis on its data.

Line Chart

COVID-19 Confirmed Cases vs Deaths

This line chart displays the relationship between the number of confirmed COVID-19 cases and the number of deaths across different countries. The x-axis represents the confirmed cases, and the y-axis represents the number of deaths.

Hover over the line to view the exact data for each point.

Bubble Chart Visualization

Cases vs Deaths vs Population

This bubble chart visualizes the relationship between COVID-19 cases, deaths, and population size across different countries. Bubble size represents population.
